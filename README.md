# Student Management System

A simple Java project built to understand and practice Object-Oriented Programming (OOP) concepts.

## Project Overview

This application manages student information such as student ID, name, course, and contact number. It demonstrates how classes and objects work together in Java.

## Features

- Create Student objects
- Store student details
- Display student information
- Update student details using setters
- Access data using getters
- Demonstrates encapsulation using private variables

## OOP Concepts Used

### Class
- Student

### Object
- Student object created in Main class

### Constructor
- Used to initialize student details

### Encapsulation
- Student data is kept private
- Accessed through getters and setters

### Methods
- displayStudent()
- getId()
- setId()
- getName()
- setName()
- getCourse()
- setCourse()
- getNumber()
- setNumber()
