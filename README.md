# Student Management System

A simple Java project developed to practice Object-Oriented Programming (OOP) concepts.

## Features
- Create Student objects
- Store student details
- Display student information
- Use constructors for initialization
- Implement getters and setters
- Demonstrate encapsulation

## OOP Concepts Used
- Classes and Objects
- Constructors
- Encapsulation
- Methods
- Getters and Setters

## Technologies Used
- Java
- VS Code
